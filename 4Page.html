Popkoor!

Dit was een heel erg recent project. Dit project was bedoeld voor een doelgroep die al een hele lange tijd een nieuwe website nodig hadden, en die gingen wij voor hun maken! Het begon allemaal natuurlijk weer met doelgroep onderzoek. Wij kwamen uiteindelijk bij het gedeelte van testen en liepen tegen heel erg veel dingen aan, vooral omdat we moeite hadden met het programmeren, dus ons product zag er ook niet helemaal geweldig uit maar het werktte wel!

Met de website kon een user inloggen, maar niet zomaar inloggen: Je kon inloggen als een koorlid of als een admin, per rol had je verschillende rechten binnen de applicatie, zo kon een admin tekst aanpassen en foto's pushen, iets wat een normaal koorlid niet kon. Ook hadden we de basics in een website gedaan; je kon nu veilig inloggen, je kan de website als admin aanpassen en de website zag er een stuk frisser en moderner uit.

Persoonlijk vond ik dit 1 van de vagere projecten omdat het heel erg serieuzer en nieuwer aanvoelde en het was het eerste project van het 2e jaar. Ik had een rol op mij genomen die niet bij mij pastte, ik moest namelijk de leiders rol oppakken. In principe was dit te doen maar ik voelde mij een beetje opgesloten. Ik wilde alles in mijn eentje doen en dus liep het niet zo soepel. Wat wel mooi is is dat dit project het project is waarvan ik het meeste heb geleerd, en dat is iets wat ik persoonlijk echt het aller belangrijkste is!

Helaas is dit project niet verder gebruikt en heel veel is helaas verloren gegaan, ik heb geeneens meer foto's! Maar ik moest dit project echt benoemen, ik heb zo veel geleerd gedurende dit project en ik heb dit alles meegenomen naar mijn volgende projecten. Zo heb ik bijvoorbeeld geleerd hoe een ondernemende technoloog tewerk gaat met een business model canvas en waardeproposities etc. Dit vind ik heel erg belangrijk voor de setup van een project en dus heeft dit mij heel erg geholpen.
